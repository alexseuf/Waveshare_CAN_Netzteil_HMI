# Build- und Abnahmecheckliste V9.1

- [ ] PioArduino 55.03.39 wird verwendet.
- [ ] Plus-/Minus-Tasten sind blau.
- [ ] Preset-Tasten sind blau und geometrisch mittig.
- [ ] START ist grün, STOP ist rot.
- [ ] Kommunikationsfehlerpunkt wird bei Timeout rot.
- [ ] Nach Wiederherstellung der CAN-Verbindung wird sofort ein Heartbeat gesendet.
- [ ] Der Kommunikationsfehler erlischt nach dem nächsten fehlerfreien Statusframe.
- [ ] CAN-Monitor zeigt 20 getrennte Zeilen ohne Überlappung.
- [ ] Debug-Log zeigt 20 getrennte Zeilen ohne Überlappung.
- [ ] Debugging erfolgt über UART GPIO43/44.
- [ ] RS485-Umbauhinweis und Foto sind vorhanden.
