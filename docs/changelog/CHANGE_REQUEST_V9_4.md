# V9.4 Änderungen

## Bereits funktionsfähig

- Einheitliche Zeitstempel im Format HH:MM:SS.
- Die zuvor für die Hauptansicht, Statusbereiche und Bedienelemente festgelegten UI-Anpassungen.

## Scope-Anpassungen

- Achsen per Zwei-Finger-Geste vergrößern und verkleinern.
- X-Achsen-Zoom verändert das sichtbare Zeitfenster.
- Zoom über einer Y-Achse verändert nur Spannung, Strom oder Leistung.
- Zoom innerhalb der Plotfläche verändert alle Achsen gemeinsam.
- Achsenbeschriftungen werden passend zur neuen Skalierung aktualisiert.
- Legende so begrenzen, dass sie sich nicht mit Achsenbeschriftungen überschneidet.
- Kurze farbige Striche zur eindeutigen Signalzuordnung in der Legende:
  - Grün: Spannung U
  - Blau: Strom I
  - Orange: Leistung P
