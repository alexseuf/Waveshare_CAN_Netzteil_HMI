# V9.3 Änderungen

- Spannungssollwert: 51,2 bis 58,4 V.
- Hauptseite: breiteres Spannungsfeld, Stromfeld bündig rechts.
- START/STOP: größere Symbole, größerer Abstand, gesamte Touchfläche, sofortige Auswertung beim Drücken.
- Scope: größere Plotfläche, X-/Y-Achsen, relative Zeitachse bis 0 s, verschiebbare Legende, Standardansicht.
- Pinch-Zoom: über X-Achse nur Zeit, über jeweiliger Y-Achse nur diese Skala, im Plot alle Achsen.
- CAN-Recovery unverändert, da V9.2 bestätigt stabil arbeitet.
