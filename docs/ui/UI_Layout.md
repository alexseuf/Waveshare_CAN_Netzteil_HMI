# UI-Layout V9 (800 × 480)

Die Hauptseite besteht aus Sollwert-, Istwert-, Preset-, CC/CV-, Energie-, Systemstatus- und Bedienbereichen. Alle Koordinaten beziehen sich auf die reale Displayauflösung 800 × 480.

- Navigation: 48 px hoch; Zahnrad rechts, 72 px breit.
- Inhaltsbereich: 780 × 422 px.
- Große Touchflächen: Plus/Minus 58 × 68 px, START/STOP 275 × 64 px.
- Preset-Tasten: 68 × 78 px; horizontal und vertikal zentriert.
- CAN-Monitor: 16 sichtbare Zeilen.
- Debug-Log: 18 sichtbare Einträge, feste Zeilenhöhe, kein Überlappen.
